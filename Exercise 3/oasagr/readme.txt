--------
ΓΕΝΙΚΑ
--------
Απο τα mockups αλλάξαμε την ιδέα των tabs.
Προσθέσαμε breadcrumb δυναμικό για κάθε σελίδα.
Χρησιμοποιήσαμε leaflet.js 1.6 για να εμφανίσουμε χάρτη.
Αν ο χρήστης δεν έχει συνδεθει μπορεί να δει τα πάντα.
Ο συνδεδεμένος έχει τα αγαπημένα του, επίσης οι υπολογισμοί γίνονται με το τύπο επιβάτη που τους αντιστοιχεί και τέλος μπορούν να αγοράσουν μειωμένα εισιτήρια άμα το δικαιούνται.


--------
ΤΕΧΝΙΚΑ
--------
Φάκελοι: components - περιέχουν τα επιμέρους κομμάτια του web app
         styles - περιέχουν τα styles.css καθως και τα JS
         images - περιέχουν κάτι εικόνες που δεν αποτελούν κομμάτι των components
         scripts - scriptακια όπως το db-connector.php


Βάση δεδομένων: χρησιμοποιήσαμε mariadb
                3 tables: users(χρήστες)
                          fav(αγαπημένα χρηστών)
                          news(νεα)
                          stops(στάσεις)
                          buses(λεωφορεία)
